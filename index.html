The FourSight Creative Problem-Solving Process

Go to Clarify Stage                 Go to Ideate Stage                Go to Develop Solutions Stage                 Go to Implement Action Plan Stage


  • This website is a guide to the creative problem-solving process for Buffalo State students taking CRS 559--Principles In Creative Problem Solving, and CRS 610--Facilitation of Group Creative Problem Solving.

  • You will find links to other websites below. If you go to any of them, click on your Internet browser's Back Button to return to this website.

  • The YouTube videos on each page are shared with the permission of Dr. Roger Firestien, Buffalo State Professor Emeritus, and can also be found on Dr. Fierstien's website.

  • At the end of each stage, you will find a link to an example of how that stage should look when completed on the virtual whiteboard/sticky note app Mural.

  • You can see how students are guided through CPS activities in Mural on this Google Doc CPS Mural Completion Guide.
The FourSight CPS Process Consists of Four Stages:
  1. Clarify the Goal, Challenge, Problem, or Opportunity
  2. Generate Ideas
  3. Develop Solutions
  4. Implement an Action Plan
